6.1 Date类

  • Date类在java.util包中，使用Date类的无参数构造方法创建的对象可以获取本地当前时间，表示时间的默认顺序是：星期,月,日,小时,分,秒,年。
  • 使用DataFormat的子类SimpleDateFormat的常用构造方法public SimpleDateFormat(String pattern)可以用参数pattern指定的格式创建一个对象,该对象调用format(Data date)方法格式化时间对象date。在pattern中有：
    • 月份：M或MM——用两位数字或文本输出；连续包含至少3个M——用汉字输出
    • 年份：y或yy——用两位数字输出；yyyy——用四位数字输出
    • 日：d或dd——用两位数字输出
    • 小时：H或HH——用两位数字输出
    • 分:m或mm——用两位数字输出
    • 秒：s或ss——用两位数字输出
    • 星期：E——用字符串输出

6.3 Math类

java.lang包中的Math类包含许多用来进行科学计算的类方法(可以直接通过类名调用)以及两个静态常量E和PI。

常用方法

  • public static long abs(double a)——返回a的绝对值
  • public static double max(double a,double b)——返回a,b的最大值
  • public static double min(double a,double b)——返回a,b的最小值
  • public static double random()——产生一个0到1之间的随机数(不包括0和1)
  • public static double pow(double a,double b)——返回a的b次幕
  • public static double sqrt(double a)——返回a的平方根
  • public static double log(double a)——返回a的自然对数(lna)
  • public static double sin(double a)——返回正弦值.
  • public static double asin(double a)——返回反正弦值.

格式化

我们可以使用java.text包中的NumberFormat类对输出的数字结果进行必要的格式化,该类调用类方法public static final NumberFormat getInstance()实例化一个NumberFormat对象,该对象调用public final String format(double number)方法可以格式化数字number.

NumberFormat的常用方法

  • public void setMaximumFractionDigits(int newValue)——设置最大小数位数
  • public void setMinimumFractionDigits(int newValue)——设置最小小数位数
  • public void setMaximumIntegerDigits(int newValue)——设置最大整数位数
