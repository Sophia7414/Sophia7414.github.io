3.1 运算符与表达式

算术运算符与算术表达式

加减运算符(+,-)

双目运算符(连接两个操作元的运算符)，结合方向从左到右，操作元是整型或浮点型数据，优先级是4级。

乘，除和求余运算符(*,/,%)

双目运算符,结合方向是从左到右，操作元是整型或浮点型数据，优先级是3级。

自增，自减运算符(++,—)

单目运算符，无相对操作元的固定位置，操作元必须是一个整型或浮点型变量，作用是使变量的值增1或减1。
位置的区别：以++为例，++x是先执行x=x+1再使用x的值，而x++是先使用x的值再执行x=x+1

算术混合运算的精度

关系运算符与关系表达式

用来比较两个值的关系，运算结果是boolean型，当对应的关系成立时结果是true,否则是false

逻辑运算符与逻辑表达式

包括&&,||,!。其中&&,||为二目运算符，实现逻辑与，逻辑或 ；!为单目运算符，实现逻辑非。逻辑运算符操作元必须是boolean型数据,可以用来连接关系表达式。逻辑运算符”&&”和”||”也称做短路逻辑运算符(前者有一个false就直接得出结果为false；后者有一个true就直接得出结果为true)

赋值运算符与赋值表达式

赋值运算符=是双目运算符，左面的操作元必须是变量。

位运算符

我们知道整型数据在内存中以2进制的形式表示，这样我们就可以对整型数据进行按位的运算。

“按位与”运算符

“&”是双目运算符，对两个整型数据a,b按位进行运算。如果a,b两个数据对应位都是1,则结果的该位是1,否则是0.如果b的精度高于a,那么结果的精度和b相同。

“按位或”运算符

“|”是双目运算符，对两个整型数据a,b按位进行运算，运算法则是如果a,b两个数据对应位都是0,则结果的该位是0,否则是1。如果b的精度高于与a,那么结果结果的精度和b相同。

“按位非”运算符

“~”是单目运算符，对一个整型数据a按位进行运算，运算法则是如果a对应位都是0,则结果的该位是1,否则是0。

“按位异或”运算符

“^”是双目运算符，对两个整型数据a,b按位进行运算，运算法则是如果a,b两个数据对应位相同，则结果的该位是0,否则是1.如果b的精度高于与a,那么结果的精度和b相同。
位运算符和逻辑运算符：设x=1，经过((y=1)=0))&&((x=6)=6))后x=1，而经过((y=1)=0))&((x=6)=6))后x=6

instanceof 运算符

双目运算符，左面的操作元是一个对象 右面是一个类.当左面的对象是右面的类创建的对象时，该运算符运算的结果是true,否则是false。

运算符综述

Java的表达式就是用运算符连接起来的，符合Java规则的式子。运算符的优先级决定了表达式中运算执行的先后顺序。

Java语言的控制语句

Java语言的控制语句有条件语句和switch开关语句两类。

条件控制语句

条件语句可分为3种：

  • if语句：
    if语句的一般形式为： 括号内的表达式的值必须是boolean类型，当值为true时，则执行紧跟着的复合语句；如果表达式的值为false,则执行程序中if语句后面的其他语句。复合语句如果只有一个语句时{}可以省略不写
  • if-else语句 括号内的表达式的值必须是boolean类型，当值为true时，则执行紧跟着的复合语句；如果表达式的值为false,则执行else后面的复合语句

在循环体中使用语句break和语句continue

在一个循环中，如果在某次循环体的执行中执行了break语句，那么整个循环语句就结束；如果在某次循环体的执行中执行了continue语句，那么本次循环就结束，而转入进行下一次循环。
