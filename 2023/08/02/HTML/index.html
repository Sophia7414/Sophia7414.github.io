01 Web

Web（World Wide Web）

全球广域网，也称为万维网,web端就是指网页端

网页

构成网站的基本元素

浏览器

网页运行的平台

Web标准

制作网页要遵循的规范。Web标准包括三个方面：
1、结构标准（HTML）：用于对网页元素进行整理和分类
2、表现标准（CSS）：用于设置网页元素的版式、颜色、大小等外观样式
3、行为标准（JS）：用于定义网页的交互和行为
可以根据web标准将web前端分为HTML（超文本标记语言）、CSS(层叠样式表)和JavaScript。

02 浏览器

浏览器的组成

1、渲染引擎（浏览器内核）

浏览器所采用渲染引擎来解析HTML与CSS。渲染引擎的作用是读取网页内容，计算网页的显示方式并显示在页面上。它决定了浏览器如何显示网页的内容以及页面的格式信息，是浏览器兼容性问题出现的根本原因。

2、JS引擎(JS解释器)

用来解析网页中的JavaScript代码，对其处理后再运行.

浏览器工作原理

浏览器主要由一下部分组成：

  • UI界面：浏览器主窗口之外的其他部分
  • 浏览器引擎：用来查询和操作渲染引擎，是UI界面和渲染引擎之间的桥梁。
  • 渲染引擎
  • 网络模块：用于发送网络请求
  • JavaScript解析器
  • UI后端：用于绘制组合框、弹窗等窗口小组件，会调用操作系统的UI方法。
  • 数据存储模块

    03 初识HTML

    概念

    全称为 HyperText Markup Language，译为超文本标记语言,是一种描述性的标记语言。HTML格式的文件是一个纯本文文件，用一些标签来描述语义
  • 超文本：超出了文本的限制的内容（图片、音频、视频等）；超级链接文本
  • 标记语言：标记语言是一套标记标签。编程语言是有编译过程的，而标记语言没有编译过程。

XHTML

可扩展超文本标注语言，是严格的、纯净的HTML。

HTML结构

HTML标签通常是成对出现的（双边标记），也有少部分单标签（单边标记）

骨架标签

  • ——页面中最大的标签，我们成为根标签
  • ——文档的头部，head标签中必须要设置title
  • ——文档的标题，让页面拥有一个网页标题
  • ——文档的主体，页面内容基本都是放到body里面的

    文档声明头

    以<!DOCTYPE ……>开头的语句，是任何一个标准的HTML页面的第一行，简称DTD，用来告知浏览器文档使用哪种HTML或XHTML规范。在HTML4.01中共有两种大规范（普通和XHTML）以及3种小规范，一共6种DTD。其中的3种小规范为：
  • strict：表示“严格的”，这种模式里面的要求更为严格
  • Transitional：表示“普通的”，这种模式没有别的规范。
  • Frameset：表示“框架”，在框架的页面使用

页面语言(lang)

<html lang="en">标签用于指定页面的语言类型，最常见的有两种：

  • en：页面语言为英语。
  • zh-CN：页面语言为中文。

头标签内部的常见标签有：

<title>

指定整个网页的标题，在浏览器最上方显示，有助于SEO搜索引擎优化。

<base>

为页面上的所有链接规定默认地址或默认目标。

<meta>

提供有关页面的基本信息,常见的几种meta标签如下:

  • charset(字符集) 表示即网页的编码方式。计算机要准确的处理各种字符集文字，需要进行字符编码，以便计算机能够识别和存储各种文字。如果meta写的和声明的不匹配，那么浏览器就是乱码。
  • viewport(视口) width=device-width:口宽度等于屏幕宽度
  • 关键字
    例： 用于提高搜索命中率，让别人能够找到你。
  • 页面描述
    例： 百度搜索结果能够显示页面描述中的语句
  • refresh(刷新)
    例： 上述标签表示3秒之后，自动跳转到百度页面。

<body>

用于定义HTML文档所要显示的内容，也称为主体标签。

<link>

定义文档与外部资源的关系。

<body>标签

属性：

  • bgcolor——设置整个网页的背景颜色。
  • background——设置整个网页的背景图片。
  • text——设置网页中的文本颜色。
  • leftmargin——网页的左边距。
  • topmargin——网页的上边距。
  • rightmargin——网页的右边距。
  • bottommargin——网页的下边距。

该标签还有一些其他的属性，例如：link属性表示超链接的默认颜色；alink属性表示鼠标点击但是还没有松开时的颜色；vlink属性表示点击完成后的颜色。

计算机编码

中文能够使用的字符集有UTF-8和GBK

  • 用meta标签声明的当前这个html文档的字库，一定要和保存的文件编码类型一样，否则乱码
  • 保存大小：UTF-8(字多，有各种国家的语言)>gb2312(字少，只用中文和少数外语和符号)

HTML的规范

  • 标签不能交叉嵌套
  • 标记都必须小写
  • 标签都必须闭合
  • 属性值必须加引号
  • 属性必须有值
  • XHTML文档开头必须要有DTD文档类型定义
  • HTML的文字之间如果有空格、换行、tab，都将被折叠为一个空格显示

拓展

  • 在VSCode中，可以通过!或html:5+tab的方式自动生成html骨架
