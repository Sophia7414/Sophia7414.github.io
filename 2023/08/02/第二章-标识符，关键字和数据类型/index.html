JAVA 第二章 标识符，关键字和数据类型 笔记

Posted on 2023-08-02 Edited on 2023-08-06

标识符

简单地说，标识符就是一个名字。Java语言规定标识符由字母，下划线，美元符号和数字组成。Java所谓的字母包括了世界上任何语言中的”字母表”,因此，Java所使用的字母不仅包括通常的拉丁字母、汉字、片假名、平假名、朝鲜文以及其他许多语言中的文字.

关键字

关键字就是Java语言中已经被赋予特定意义的一些单词.不可以把这类词作为名字来用Java的关键字有：
abstract，boolean，break，byte，case，catch，char，class，continue，do，double，else，extends，false，find，finally，float，for，implements，import，instanceof，int，interface，long，nativenew，null，package，private，public，return，short，static，super，switch，synchronized，this，throw，true，try，void，while。

基本数据类型

逻辑类型

常量：true,false

整数类型

常量:十进制、八进制、十六进制
整型变量(分为4种):

int型

使用int来定义


int型变量内存分配给4个字节，正数使用原码表示，最高位是0；负数用补码表示，最高位是1。

byte型

使用byte来定义


byte型变量内存分配给1个字节,因此取值范围是：$-2^7$~$2^7-1$

short型

使用short来定义


short型变量内存分配给2个字节，因此取值范围是：$-2^{15}$~$2^{15}-1$

long型

使用long来定义


long型变量内存分配给8个字节,因此取值范围是:$-2^{63}$~$2^{63}-1$

字符类型

常量:unicode字符集，共有65535个。
变量：
使用char来定义


char型变量内存分配给2个字节,没有负数,取值范围是0~65536.对于char x='a';，内存x中存储的是97,(字符a在unicode表中的排序位置)char x=97;。要观察一个字符在unicode表中的顺序位置，必须使用int类型显示转换；如果要得到一个0~65536之间的数所代表的unicode表中相应位置上的字符也必须使用char型显示转换。

浮点类型 实型

浮点型分两种:

float型

使用float来定义


float型变量内存分配给4个字节，取值范围：$10^{-38}$~$10^{38}$,$-10^{38}$~$-10^{-38}$

double型

变量:
使用double来定义


double型变量内存分配给8个字节,取值范围大约是$10^{-308}$~$10^{308}$
